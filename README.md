# Mi Último Adiós

"A veces, programar es la mejor forma de procesar lo que el corazón no entiende. Este pequeño sitio es el punto final de una historia que guardaré con cariño, pero que hoy decido dejar de escribir."

### Sobre este proyecto:
Este es un proyecto personal de cierre. Una página web interactiva diseñada para expresar sentimientos de gratitud y despedida, utilizando:
* HTML5
* CSS3 (Animaciones suaves)
* JavaScript (Efecto de cierre)

*Siempre te tendré en mi recuerdo, Liz.*
